## Überschriften

*Markdown* bietet zwei verschiedene Möglichkeiten, Überschriften auszuzeichnen.

### Möglichkeit 1

Überschrift ersten Grades (`<h1>`)

```
Überschrift ersten Grades
=========================
```

Überschrift zweiten Grades (`<h2>`)

```
Überschrift zweiten Grades
--------------------------
```

### Möglichkeit 2

```
# Überschrift ersten Grades
```

```
## Überschrift zweiten Grades
```
